# Легковесное локальное веб-приложение для интервального повторения знаний из Markdown-файлов Obsidian.

Читает только `.md` из папки в корне хранилища Obsidian (родительская папка от `learn-app`). Карточки: заголовок — «вопрос», текст до следующего заголовка того же или более высокого уровня — «ответ». Ссылки `[[...]]` на обороте показываются без скобок.

## Запуск

1. открыть терминал в папке `learn-app`.
2. выполни
```bash
python server.py
```
3. В браузере открыть: http://127.0.0.1:8765

Если по `localhost:8765` страница не грузится, оставьте **127.0.0.1** — на части систем `localhost` уходит на IPv6, а сервер слушает только IPv4.

### Навык и подкаталог

- **Навык** - это прямая подпапка внутри `основного хранилища` (например `Английский`, `Немецкий` или любая изучаемая область). Список строится автоматически по именам папок.
- **Подкаталог** — необязательно: первый уровень внутри выбранного навыка, где есть хотя бы одна `.md` заметка (в этой папке или глубже). Значение **«Весь навык»** включает все заметки навыка.

Выбранные навык и подкаталог сохраняются в браузере (`localStorage`).

Ответ на карточке рендерится как **Markdown** через скрипты с CDN (`marked` + `DOMPurify`). Нужен интернет при первой загрузке страницы.

## Как пользоваться

- **Показать ответ** - раскрыть текст под заголовком.
- **Помню** - следующая случайная карточка (после конца колода снова перемешивается).
- **Повторить** - эта карточка уходит в конец очереди.
- **Фильтр** - оставить только карточки, у которых в пути файла или во «вопросе» есть введённая подстрока (без учёта регистра). Enter или «Обновить колоду» - вносит изменения.
