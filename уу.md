# Learn-App

Легковесное локальное веб-приложение для интервального повторения знаний из Markdown-файлов.

Читает только `.md` из папки "Знания" в корне хранилища (родительская папка от `learn-app`).

---

## Запуск

python server.py

Открыть в браузере: http://127.0.0.1:8765

Используйте именно 127.0.0.1, не localhost.

---

## Структура заметок

#### 1.1 Основной вопрос
Текст ответа.

##### 1.1-1 Подвопрос
Текст ответа.

#### - основные вопросы
##### - подвопросы

Номер раздела в пути файла должен совпадать с нумерацией в заголовках.

---

## Фильтрация

Фильтр оставляет только карточки из файлов, в пути которых есть введённая подстрока.

Примеры:
1. -> все карточки из файлов с 1. в пути (Базы данных)
2. -> все карточки из файлов с 4. в пути (ООП)

Только основные вопросы - при включении чекбокса в колоду попадают только карточки уровня 4 (####). Подвопросы уровня 5 (#####) отсеиваются.

---

## Пример работы

Фильтр 4., чекбокс выключен -> Все карточки раздела 4 (основные + подвопросы)
Фильтр 4., чекбокс включён -> Только основные вопросы раздела 4

---

## Как пользоваться

Показать ответ - раскрыть текст под заголовком.
Помню - следующая случайная карточка.
Повторить - карточка уходит в конец очереди.
Фильтр - введите номер раздела и нажмите Enter или Обновить колоду.
Только основные вопросы - включите чекбокс, чтобы убрать подвопросы.

---

## Хранение настроек

Навык, подкаталог и состояние чекбокса сохраняются в localstorage и восстанавливаются при перезагрузке.
