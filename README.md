## 🧠 Система обучения/ технические детали
### Learn-App

- **Запуск:** `python server.py` в папке `learn-app`
- **Адрес:** http://127.0.0.1:8765 (не localhost!)
- **Остановка:** `Ctrl+C` в терминале
- **Рендеринг:** Markdown через `marked` + `DOMPurify` (нужен интернет
### Фильтрация

Каждый раздел программы имеет **уникальный номер** (1. - 8.), который проставляется:
- В начале названия файла (например: `1. Основы информатики.md`)
- В заголовках внутри файлов (например: `# 1.1. Что такое информатика`)

Для изучения выбранного раздела необходио определить его номер в фильтре, пример "1." или "3." после чего нажать "**Обновить колоду**"
